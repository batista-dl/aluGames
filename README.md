# aluGames

Projeto desenvolvido como parte do curso de **Lógica de Programação** da Alura, com foco em **JavaScript**.  

O HTML e CSS vieram fornecidos pelo curso. O objetivo deste projeto foi trabalhar a lógica de programação e interatividade usando JavaScript.

---

## Sobre o Projeto

O aluGames é um **dashboard de jogos** que permite:

- Visualizar todos os jogos disponíveis  
- Indicar quais jogos estão alugados  
- Interagir com os botões de alugar/devolver  
- Layout responsivo para desktop e celular

---

## Tecnologias Utilizadas

- JavaScript – lógica e interatividade  

## Funcionalidades

- Contagem automática de jogos alugados  
- Overlay visual para jogos alugados  
- Botões interativos para alugar/devolver  
- Interface adaptada para dispositivos móveis
